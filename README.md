# Clinical Management System - Frontend

This project is the frontend interface for a Clinical Management System developed using HTML, CSS, and JavaScript.

The system provides a simple interface for hospital staff such as lab technicians to navigate dashboards, manage tests, and handle billing operations.

## Features
- Homepage with hospital departments
- Login page for system access
- Lab technician dashboard
- Test management section
- Billing page

## Technologies Used
- HTML
- CSS
- JavaScript

## Future Improvements
- Backend implementation using Django
- Full-stack application using React and Django
